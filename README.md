# Build instruction
- Create 2 directories: `deps` and `build`
- Enter `deps` to <a href="https://github.com/yourWaifu/sleepy-discord">sleepy-discord</a>, then go back
- `cmake -S . -B ./build`
- `make -C build`

The executable `colorole` will appear.
